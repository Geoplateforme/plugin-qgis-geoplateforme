- Description :

Suppression des accès associé à une clé utilisateur.

- Paramètres :

| Entrée           | Paramètre          | Description                                                |
|------------------|--------------------|------------------------------------------------------------|
| Identifiant de la clé    | `KEY_ID`        | Identifiant de la clé.  |

Nom du traitement : `geoplateforme:delete_user_key_accesses`
